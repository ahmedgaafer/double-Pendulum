How to use?

  • Change the values of the starting position of the pendulm.
  • Click "Start simulation" button to start.
  • Click the "Play/Pause" button to pause and play the simulation.

Input description:

  1. Length 1 : length of the first "red" string range : (50 : 200)
  2. Length 2 : length of the second "blue" string range : (50 : 200)
  3. Mass 1: weight of the "red" ball range(0.2:20)
  4. Mass 2: weight of the "blue" ball range(0.2:20)
  5. Angle 1 : Angle of "blue" string range(0:360)
  6. Angle 2 : Angle of "red" string range(0:360)
  7. Gravity : the gravity you wish to apply
  8. Friction: the loss ratio. range: (0 : 1)
    ex: if friction is 0.998 this means that the speed decreased by 0.998 per frame
  9. Speed of simulation: the speed of the running simulation range: (0 : 60)

Notes:

  • Chnaging the values will RESET the simulation.
  • Clicking the "Start simulation" button will RESET the simulation
  • Please stay within the described limits above to have a good experince watching the simulation
